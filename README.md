# CRM Sales Opportunities Analysis Using SQL & Power BI

## About This Project

This project analyzes B2B sales pipeline data for a fictional company that sells computer hardware. The goal is to understand sales team performance, product success, and identify areas for improvement. Essentially, it’s about turning raw sales data into actionable insights.  

**Data Source:** [Kaggle](https://www.kaggle.com/datasets/innocentmfa/crm-sales-opportunities)

## Objectives

The main goals of this project are:  

- **Sales Team Performance:** Identify which teams are excelling and which need improvement.  
- **Sales Agent Productivity:** Spot underperforming sales agents and provide insights for coaching.  
- **Trends Analysis:** Detect quarter-over-quarter trends in sales performance.  
- **Product Success Rates:** Find which products close deals more effectively.  

**Personal Goals:** Improve SQL skills with CTEs and Window Functions, enhance DAX measures, and build dynamic dashboards in Power BI.  

## Project Workflow

### 1. Data Preparation
- **Exploring the Data:** Examined raw files to understand structure and contents.  
- **Data Cleaning & Transformation:** Prepared the data for analysis.  

### 2. Data Modelling
- **Schema Design:** Defined relationships between tables for efficient querying.  
- **Database Setup:** Created tables in SQL and loaded prepared data.  

### 3. Data Analysis
- **SQL Queries:** Explored data to answer core business questions.  
- **Focus Areas:**  
  - Team Performance  
  - Underperforming Agents  
  - Quarterly Trends  
  - Product Win Rates  
  - Sector Performance  
  - Sales Cycle Duration  

### 4. Reporting
- **Power BI Dashboard:** Built an interactive report to visualize insights.
